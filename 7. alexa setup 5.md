# Set up the microphone

<H3> 1. Configure the " microphone & speaker" </H3>
<H4> &nbsp;&nbsp;&nbsp;  Create & Open .asoundrc </H4>

<PRE>
 <CODE>
    nano ~/.asoundrc
 </CODE>
</PRE>

<H4> &nbsp;&nbsp;&nbsp;  Add the line </H4>

<PRE>
 <CODE>
  pcm.!default {
    type asym
    playback.pcm {
      type plug
      slave.pcm "hw:0,0"
    }
    capture.pcm {
      type plug
      slave.pcm "hw:1,0"
    }
  }
 </CODE>
</PRE>
